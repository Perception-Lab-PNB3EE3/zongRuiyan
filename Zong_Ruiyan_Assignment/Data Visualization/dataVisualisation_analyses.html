# First visualize the whole dataset
y_values = df["y"].values
x_values = df["x"].values

conditions = df["condition"].values
unique_conditions = np.unique(conditions)

COLORS = ["black", "#D95F02", "#7570B3", "#91BA88"]

fig, ax = plt.subplots(figsize=(8,8))
for condition, color in zip(unique_conditions, COLORS):
    idxs = np.where(condition == conditions)
    # No legend will be generated if we don't pass label=species
    ax.scatter(
        x_values[idxs], y_values[idxs], label=condition,
        s=50, color=color, alpha=0.7
    )
    
ax.legend()
plt.show()

What can be concluded from the dataset?¶

Why is it important to visualize data?¶

People are usually not as sentive to the numerical data as the visual color and patterns. Visualizing the data helps people obtain more information, such as the trends, correlations and interactions between different conditions. It also makes the scientific communication easier because researchers doesn't have to show all the numerical data to explain their result, instead, a plot will be straightforward and easy to understand.
